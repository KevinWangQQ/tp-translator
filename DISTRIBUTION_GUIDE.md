## 📋 用户安装流程

### 用户需要做的：
1. **下载**: 获取 `cube-translator-v1.1.0.zip`
2. **解压**: 解压到本地文件夹
3. **导入**: Figma Desktop → Plugins → Development → Import plugin from manifest
4. **配置**: 获取OpenAI或Gemini API密钥并在设置中配置
5. **使用**: 选择文本 → 翻译

### 系统要求：
- Figma桌面应用（必须）
- OpenAI或Google Gemini API密钥
- 稳定的网络连接
